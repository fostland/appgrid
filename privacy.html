Доступно в RuStore

Коротко о главном: приложение «Блокнот ЖКУ» не собирает, не передаёт и не продаёт ваши персональные данные. Все данные, которые вы вводите, хранятся исключительно на вашем устройстве в зашифрованном виде.

01

Общие положения

Настоящая политика конфиденциальности описывает, как мобильное приложение «Блокнот ЖКУ» (далее — «Приложение») обрабатывает информацию пользователя. Используя Приложение, вы соглашаетесь с условиями данной политики.

Разработчик Приложения — индивидуальный разработчик, публикующий приложения на сайте appgrid.ru.

02

Какие данные хранит приложение

Приложение хранит следующие данные, которые вы вводите вручную:

  • Показания счётчиков — значения воды, газа, электричества и других ресурсов
  • Тарифы — стоимость единицы ресурса, которую вы указываете самостоятельно
  • История платежей — записи о расчётах за прошлые периоды
  • Порталы оплаты — названия и ссылки на сайты оплаты ЖКУ, которые вы добавляете
  • Настройки приложения — выбранная тема, параметры отображения

Все перечисленные данные хранятся только на вашем устройстве. Разработчик не имеет к ним доступа.

03

Чего приложение не делает

  • Не передаёт данные на сторонние серверы
  • Не собирает аналитику, статистику использования или сведения об устройстве
  • Не использует рекламные трекеры и SDK
  • Не запрашивает доступ к контактам, геолокации, микрофону или файлам устройства
  • Не создаёт учётную запись пользователя и не требует регистрации
  • Не выполняет сетевых запросов в фоновом режиме

04

Запрашиваемые разрешения

Приложение запрашивает только те разрешения, которые необходимы для работы заявленных функций:

📷 CAMERA
Только для сканирования QR-кодов квитанций СБП. Не требуется обязательно.
🌐 INTERNET
Только для открытия порталов оплаты ЖКУ, которые вы сами добавляете.
📳 VIBRATE
Тактильная обратная связь при успешном сканировании QR-кода.

Разрешение на камеру запрашивается только в момент использования функции сканирования и не является обязательным для работы Приложения.

05

Защита данных

Все данные приложения хранятся в зашифрованной базе данных с использованием SQLCipher. Ключ шифрования генерируется на устройстве и хранится в Android Keystore — защищённом хранилище операционной системы, недоступном для других приложений.

Это означает, что даже при физическом доступе к устройству или файлам приложения данные не могут быть прочитаны без разблокировки устройства.

06

Резервные копии и передача данных

Приложение не создаёт и не управляет облачными резервными копиями. Резервное копирование данных приложения отключено на уровне манифеста.

Перенос данных между устройствами средствами Приложения не предусмотрен.

07

Дети

Приложение не предназначено для детей младше 6 лет и не направлено специально на детскую аудиторию. Приложение не собирает никаких данных ни от кого, включая детей.

08

Изменения политики

Если политика конфиденциальности будет изменена, обновлённая версия появится на этой странице (appgrid.ru/privacy.html) с новой датой вступления в силу. Мы рекомендуем периодически проверять эту страницу.

Продолжение использования Приложения после изменений означает согласие с обновлённой политикой.

09

Удаление данных

Чтобы полностью удалить все данные приложения — удалите Приложение со своего устройства. После удаления все данные, включая историю показаний и настройки, будут удалены без возможности восстановления.
